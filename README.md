# wechat-me 微信机器人

A wechat client with built in robot. 个人微信客户端+机器人。

[![Node version](https://img.shields.io/badge/node-_7.6.0-green.svg?style=flat)](http://nodejs.org/download/)

## Getting Started
 - git clone https://github.com/jaypeng2015/wechat-me
 - cd wechat-me
 - npm install
 - npm install -g yarn
 - yarn
 - yarn start
